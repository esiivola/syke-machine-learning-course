Step 1: Create an new anaconda environment and open it in terminal. Navigate to this folder

Step 2: Install required anaconda packages:
> conda install pip

Step 3: Install required packages to pip
> pip install -r requirements.txt --proxy=http://gate102.vyh.fi:81

Step 4: Install cartopy:
> conda install -c conda-forge cartopy
> conda install -c conda-forge pyproj
> conda install -c conda-forge pyepsg

Step 5: add current kernel to jupyter notebook:
> conda install -c anaconda ipykernel
> python -m ipykernel install --user --name=<name of the jupyter environment>
